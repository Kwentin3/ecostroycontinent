# Change Report

Документ: Launch SEO Core Spec v0.1  
Дата: 2026-03-23

## Почему рекомендован именно такой launch-core

- Он удерживает phase 1 узким: один региональный кластер, `5-6` сильных money pages, `3-4` case pages и небольшой supporting layer.
- Он согласован с действующим каноном: RU-only, one primary region cluster, human-mediated publish, no AI chat / calculator in phase 1.
- Он опирается на практический local SEO контур для Сочи: домостроение, монолит, реконструкция и proof-led service pages выглядят более реалистично для запуска, чем широкий "строим всё для всех".
- Он оставляет честный stop-rule: слабые страницы не публикуются только ради ширины ядра.

## Что сознательно не включено в стартовый scope

- Мульти-региональная стратегия.
- Большой блог и широкий keyword map.
- Public AI chat, calculator, CRM-lite, SEO dashboard.
- Индексируемые фильтры, сложные portfolio taxonomies и прочие расширения beyond phase 1.
- Массовые service pages без proof inventory.

## Какие решения всё ещё требуют owner confirmation

- Primary launch region cluster: `Сочи / Большой Сочи` или резервный tighter focus.
- Финальный список launch money pages: выпускать ли `Строительство коммерческих объектов` сразу или отложить.
- Какие реальные кейсы и proof assets доступны для guest / commercial / reconstruction контента.
- Какие формулировки по стоимости, срокам и гарантиям компания готова публично подтверждать.
- Какие case pages считать flagship и всегда отдавать на `Business Owner review`.

## Внешние сигналы, использованные для рекомендации

- Публичная карточка компании в РБК как якорь по Сочи.
- Реестр членов СРО, где компания видна как `ООО «ЭКОСтройКонтинент»`.
- Видимые SERP-сигналы по локальным запросам `строительство домов Сочи`, `монолитные работы Сочи` и related intents.
- Локальный контекст по коммерческому строительству в Сочи, который выглядит более чувствительным и потому требует более сильной proof base.
