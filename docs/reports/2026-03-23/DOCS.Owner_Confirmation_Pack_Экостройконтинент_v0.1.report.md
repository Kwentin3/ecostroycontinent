# Owner Confirmation Pack Report

Документ: Owner Confirmation Pack v0.1  
Дата: 2026-03-23

## Какие решения уже почти подтверждены текущим каноном

- Launch остаётся `RU-only`.
- Phase 1 должен держаться одного primary region cluster.
- Human-mediated publish и owner review для claims-heavy pages уже канонизированы.
- `Строительство домов под ключ`, `Монолитные работы` и `Реконструкция / капитальный ремонт` уже выглядят как strongest default ядро.
- `Строительство коммерческих объектов` уже выглядит слабым кандидатом для phase 1 без дополнительных доказательств.

## Какие решения по-прежнему реально открыты

- Публичное имя бизнеса как brand wording на сайте.
- Финальный launch region / service area wording.
- Публичный phone / messengers / email set.
- Остаётся ли `Строительство гостевых домов` внутри phase 1.
- Какие кейсы реально становятся launch priority.
- Какие формулировки по срокам, стоимости и гарантиям owner готов публично подтверждать.

## Какие рекомендации даны по умолчанию

- Публичное имя: `Экостройконтинент`, с юридическим `ООО "ЭКОСТРОЙКОНТИНЕНТ"` в реквизитах.
- Launch region: `Сочи / Большой Сочи`; `Адлер / Сириус` только как backup.
- Launch service core: `5` страниц, без commercial page по умолчанию.
- Guest-house page: оставлять только при подтверждении реального scope.
- Primary CTA truth: `Получить консультацию и обсудить проект`.
- Claim posture: factual / careful / no bold promises without explicit owner approval.

## Что сознательно не включено в pack

- Пересказ PRD и content contract.
- Полный evidence register по каждой странице.
- Большой SEO/content plan.
- Архитектурные и implementation-level детали.
- Детальный backlog на производство контента.

Pack оставлен коротким специально: он должен закрыть owner decisions, а не породить ещё один общий spec.
